Удивительное рядом, оказывается, готового пакета Dropbox для Debiana нет. Придется собирать руками! Не беда, на форуме нашел готовый рецептик от hadret

# echo deb-src http://mentors.debian.net/debian unstable main contrib non-free >> /etc/apt/sources.list
# aptitude update
# aptitude build-deb dropbox
~ apt-get source -b dropbox
# dpkg -i dropbox_0.7.97-1_*.deb

Проверено - работает
