Доступен новый промежуточный релиз kazsid 20110225.

Внимание: Xfce версии 4.6.2. Iceweasel (Firefox) - 3.6.13.

Обещанные Xfce 4.8, Firefox 4 пока не доступны для установки, после их появления будет и новый релиз kazsid-а.

Что в комплекте?

- gstreamer0.10-plugins-bad временно исключен из комплекта из-за проблем с зависимостями;

- в ветке sid сейчас ядро linux 2.6.37, значит, улучшения;

- wine взят нестабильной ветки 1.3.х, значит, поддерживается больше программ;

- спасибо SoulFriend, теперь на борту по умолчанию новые обои на рабочий стол;

- добавлены программы Skype, Qutim, Comix, Gdebi, fontypython;

- Wicd заменен на Network Manager, теперь работать стало удобнее;

- OpenOffice.org 3.2.1 заменен на Libreoffice 3.3.1.
