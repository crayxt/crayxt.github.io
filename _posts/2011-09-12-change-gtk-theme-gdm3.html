Разобрался еще с одной проблемой в Kazsid-е, для менеджера входа в систему (GDM3) тема не применялась, то есть он просто не реагировал на изменения в своем конфигурационном файле. Оказалось, нужно было поставить один пакет

sudo apt-get install gnome-settings-daemon

который и хранит эти настройки. Почему он не ставится автоматически - потому что для apt при сборке системы задается опция --no-install-recommends, чтобы не ставило кучу лишнего.

Теперь все нормально. Значит будем пересобирать Казсид.
